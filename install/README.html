URL Shortener - Instalação

← Voltar

Pré-requisitos

  • Java 8 ou superior
  • Maven 3.6+

Compilar

mvn clean compile

Testes

mvn test

Executar

mvn jetty:run

Acessar

http://localhost:8080/

APIs

  • GET / - Página principal
  • POST /api/shorten - Encurtar URL
  • GET /api/{code} - Redirecionar
  • GET /api/info/{code} - Info
  • GET /api/health - Health
